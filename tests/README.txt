Tests for codalab are written in Selenium Driver

INSTALLATION

For installing selenium, run the following while inside your virtualenv

	venv/bin/pip -U selenium

For Google Chrome support:
	
	1. download the ChromeDriver from https://code.google.com/p/selenium/wiki/ChromeDriver
	2. move it to /usr/local/bin
	3. set appropriate permissions
		chmod 777 /usr/local/bin/chromedriver
